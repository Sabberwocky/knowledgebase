Volumes & Mounts
---

see attached volumes
```bash
lsblk
```
*create a new volumen if needed in Linode and add to VM
1. create a filesystem
```bash
 mkfs.ext4 "/dev/disk/by-id/scsi-0Linode_Volume_storage"  
```
2. create a mountpoint
```bash
mkdir "/mnt/storage"
```
3. mount automatically by adding to */etc/fstab*
```bash
/dev/disk/by-id/scsi-0Linode_Volume_storage /mnt/storage ext4 defaults,noatime,nofail 0 2
```
